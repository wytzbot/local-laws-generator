# Local Laws Generator

Generate country-specific legal document templates in 60 seconds. 100% free, runs offline, no data collection.

**Live Demo**: https://yourname.github.io/local-laws-generator

### Supported Countries
US, UK, Canada, Australia, Nigeria, India, South Africa, Kenya, Philippines, UAE

### Documents Available
- Tenancy/Lease Agreements
- Loan Agreements  
- Non-Disclosure Agreements (NDA)
- More added monthly

### Features
- Instant PDF download
- Works 100% offline after first load
- No signup, no tracking, no server
- Mobile friendly + installable as app

### How to Use
1. Pick country → document → fill form → download
2. Print and sign with witnesses if required by local law

### For Developers
This is a static HTML/JS app. No build step needed.
